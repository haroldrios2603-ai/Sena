# Manual de Usuario - Operador (RM Parking)

## Bienvenido
Este manual describe el uso del software RM Parking para los empleados que gestionan el día a día del parqueadero.

## 1. Inicio de Sesión
1. Ingrese a la URL del sistema.
2. Ingrese su correo electrónico y contraseña.
3. El sistema registrará automáticamente su entrada (asistencia).

## 2. Registro de Ingreso (RF 2.1)
1. En el panel principal, seleccione **Ingreso de Vehículo**.
2. Digite la placa del vehículo.
3. El sistema captura automáticamente la hora.
4. El sistema imprimirá el ticket con código de barras/QR (o solicitará lectura de tarjeta según configuración).

## 3. Registro de Salida (RF 2.2)
1. Seleccione **Salida de Vehículo**.
2. Escanee el código del ticket o ingrese la placa.
3. El sistema calcula automáticamente el valor a pagar basado en las tarifas configuradas.
4. Se imprime el recibo de pago.

## 4. Ocupación (RF 2.4)
1. El panel principal muestra el número de puestos libres.
2. Si el parqueadero está lleno, el sistema generará una alerta visual y bloqueará nuevos ingresos.

## Notas importantes
- El software funciona en dispositivos móviles y tablets.
- Cada acción crítica queda registrada en el historial de auditoría para su seguridad.
