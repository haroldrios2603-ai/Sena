## Descripción
Inicializar los proyectos de Frontend (Admin/Operador) usando React + Vite + TypeScript. Configurar Tailwind CSS y Router.

## Criterios de Aceptación
- [ ] Proyecto Vite creado.
- [ ] Tailwind CSS configurado y funcionando.
- [ ] React Router instalado con rutas dummy (/login, /dashboard).
- [ ] Estructura de carpetas (`components`, `pages`, `hooks`) definida.
- [ ] Linting sin errores.

## Evidencia Requerida
- Captura de pantalla del Home y Login.
- Salida de `npm run build`.
