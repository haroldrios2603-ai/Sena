# Manual de Usuario - Administrador (RM Parking)

## Bienvenido
Este manual describe las funciones avanzadas del software RM Parking destinadas a los administradores del establecimiento.

## 1. Configuración de Parqueadero (RF 1.1)
1. Ingrese con su cuenta de Administrador.
2. Vaya a la sección **Configuración > Parqueaderos**.
3. Haga clic en **Registrar Parqueadero**.
4. Complete los datos: Nombre, Dirección, Capacidad, Coordenadas GPS y Modo de Control (Tickets o Tarjetas).

## 2. Configuración de Tarifas (RF 1.2)
1. En la configuración del parqueadero, seleccione **Tarifas**.
2. Defina el valor por hora y hora adicional para cada tipo de vehículo (Carro, Moto, etc.).
3. El sistema aplicará estos valores automáticamente al registrar salidas.

## 3. Gestión de Colaboradores (RF 1.5)
1. En la sección **Usuarios**, seleccione **Colaboradores**.
2. Puede crear nuevos perfiles para sus empleados, asignándoles el rol de OPERADOR.
3. El sistema llevará el registro de auditoría de cada acción realizada por ellos.

## 4. Reportes y Estadísticas (RF 3 y 4)
1. Acceda a la pestaña de **Reportes** para visualizar:
   - Vehículos dentro y tiempo transcurrido.
   - Facturación total por periodo.
   - Asistencia de empleados.
2. Use la pestaña de **Indicadores** para ver gráficos de ingresos y horas pico.

## Notas importantes
- Como administrador, usted tiene acceso completo al historial de auditoría.
- Asegúrese de que el horario de atención esté correctamente configurado (RF 1.4).
